# Controls
You can use the following keybindings on the home page:
  - **[Space]** Pause/Play
  - **[N]** New video
  - **[M]** Open/Close menu
  - **[F]** Toggle fullscreen
  - **[Up arrow] / [Down arrow]** Change volume
  - **[Left arrow] / [Right arrow]** -/+ 5 seconds

You can also use mouse wheel to change volume.
